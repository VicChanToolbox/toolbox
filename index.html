重複清理運作原理與說明

Q1. 重複檔案可以直接在本機刪除嗎？

可以！ 本軟體深度整合了現代瀏覽器的 File System Access API。當您點擊「選擇本機資料夾 (支援直接刪除)」並取得單次授權後，即可在此網頁中不下載任何外部軟體，直接進行硬碟上的實體重複檔案清除。

Q2. 如果不小心選成唯讀/拖曳模式怎麼清理？

若您使用的是傳統唯讀模式，在勾選完成後點選「展開清理方案」，我們提供您：方案 A 打包唯一的保留檔案 (Zip) 或 方案 B 產生一鍵刪除批次指令檔 (Windows CMD / macOS Bash) 讓您在本機終端機快速執行。

Q3. 「快速比對」和「雜湊比對」有何區別？

快速比對採用「檔案名稱 + 檔案大小」的比對策略，速度極快；雜湊比對使用 SHA-256 數位指紋特徵碼，即使檔名不同、只要內容相同就能完美抓出！
